# Gathr – Notes

- Create a clear project structure showing what has been done and what still needs to be done.
- Create a dropdown/searchable box for possible activities where users can type and get suggestions.
- People must be able to rate activities.
- Add online activities (such as gaming) with separate rankings per game (e.g., CS, LoL, etc.).
- For social activities, there should be no skill-level requirement.
- During registration, collect:
  - Gender (male/female)
  - Age group (14 and under, 15–18, 19–25, 26–40, etc.)
  - Phone (for verification)
  - Email (for verification)
  - Photo
  - Full name
- Location input should support postcode suggestions/autocomplete.
- Users should also be able to drop a pin on a map for precise spots (e.g., a specific basketball court in a park).
- When creating an activity, users should be able to add a description.
- Profiles should include an “About me” description.
- Host should be able to invite people.
- Members should also be able to request/invite attendance, but host approval is still required.
- Activities should include the required number of people.
